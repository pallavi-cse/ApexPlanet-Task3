# Retail Sales Analysis Dashboard

## Objective
Perform deep-dive analysis on retail sales data and create visualizations to identify trends and business insights.

## KPIs
- Total Sales
- Total Orders

## Analysis Performed
- Sales by Category
- Sales by Region
- Top 10 Products
- Sales by Customer Segment
- Sales Trend Analysis

## Tools Used
- Python
- Pandas
- Matplotlib
- Kaggle Notebook

## Key Insights
- Identified top-performing categories and products.
- Compared sales across regions.
- Analyzed customer segments.
- Observed sales trends over time.
